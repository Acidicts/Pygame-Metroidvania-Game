# Metroid-Vania Game in Pygame
### Development on previous Versions
- Same Assets
- Same Utils
- Different Game Logic
---
### Controls
- Movement - Arrow Keys
- W to interact
- X to attack
- TAB for inventory

### Features
- Enemies
  - Ground Crawler
  - Flying
- Shop - Blue
- Dialogues - Green
- Simple Tile System

### Installation
1. Download Latest Release Demo.zip
2. Unzip Demo.zip
3. Run main.exe
